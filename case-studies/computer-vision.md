# Computer-Vision Systems

## Scope

My primary specialization is computer vision across recognition, detection, segmentation, OCR, generation, and real-time inference.

## Representative systems

| Area | Engineering focus |
| --- | --- |
| Biometric verification | Facial representations, eye detection, liveness checks, secure templates, validation workflows |
| Document intelligence | OCR, table extraction, field normalization, confidence-aware outputs |
| Virtual try-on | Body and clothing segmentation, generative models, style transfer, low-latency APIs |
| Autonomous perception | Detection, tracking, temporal context, simulation and edge-oriented inference |
| Sports and safety analytics | Player/ball tracking, drowsiness detection, keypoints, speed and distance estimation |

## Production approach

I separate data preparation, training, evaluation, inference, and application integration. Depending on the use case, I work with PyTorch, TensorFlow, Keras, YOLO, OpenCV, FastAPI, Docker, Kubernetes, and cloud or edge deployment workflows.

The goal is not only a high offline metric. I also consider latency, failure modes, monitoring, privacy, reproducibility, and how a user or downstream system will act on the prediction.
